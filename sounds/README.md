# Son de la boîte

Placez ici le fichier `boite_a_coucou.wav` joué au déclenchement.

Le son original du sketch des Guignols de l'info est protégé par le droit
d'auteur et n'est donc pas distribué avec ce projet. Fournissez votre propre
enregistrement (n'importe quel WAV convient, idéalement court).

Format utilisé pour le prototype : WAV 44100 Hz, 16 bits, stéréo.
Le chemin attendu par le code est `sounds/boite_a_coucou.wav`
(voir `play_sound()` dans `aquecoucou.py`).
